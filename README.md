# banco-preguntas-olimpiada-matematicas-sonora
Este repositorio contiene preguntas STACK en Moodle. Las preguntas abarcan 
temas de Combinatoria, Teoría de Números y Geometría, las cuales son randomizables 
en cada intento. Estas preguntas sirven como un entrenamiento temprano para la 
Olimpiada Sonorense de Matemáticas.
